Q.1 Write a menu driven program to perform the following stack
 Operations.
 i. Insert character A to F
 ii. Remove first element
 iii. Display remaining stack element. ?

ans-

 <?php
$stack = [];
function displayMenu() {
    echo "\nMenu:\n1. Insert A to F\n2. Remove first element\n3. Display stack\n4. Exit\n";
}
function insertElements(&$stack) {
    $stack = ['A', 'B', 'C', 'D', 'E', 'F'];
    echo "Inserted A to F.\n";
}
function removeElement(&$stack) {
    echo empty($stack) ? "Stack is empty.\n" : "Removed: " . array_shift($stack) . "\n";
}
function displayStack($stack) {
    echo empty($stack) ? "Stack is empty.\n" : "Stack: " . implode(", ", $stack) . "\n";
}
do {
    displayMenu();
    $choice = readline("Enter choice (1-4): ");
    switch ($choice) {
        case 1: insertElements($stack); break;
        case 2: removeElement($stack); break;
        case 3: displayStack($stack); break;
        case 4: echo "Exiting.\n"; break;
        default: echo "Invalid choice!\n";
    }
} while ($choice != 4);
?>

Q.1 Write a menu driven program to perform various file operation.
 Accept file name from user.
 a. Display type of file.
 b. Delete a file
 c. Display last access time of file ?

ans- 

 <?php
function displayMenu() {
    echo "\nMenu:\na. Display type of file\nb. Delete the file\nc. Display last access time of the file\nd. Exit\n";
}

function displayFileType($filename) {
    echo file_exists($filename) ? "The type of the file is: " . filetype($filename) . "\n" : "File does not exist.\n";
}

function deleteFile($filename) {
    echo file_exists($filename) ? (unlink($filename) ? "File '$filename' has been deleted.\n" : "Error: Could not delete the file.\n") : "File does not exist.\n";
}

function displayLastAccessTime($filename) {
    echo file_exists($filename) ? "The last access time of the file is: " . date("F d Y H:i:s.", fileatime($filename)) . "\n" : "File does not exist.\n";
}

do {
    $filename = readline("Enter the file name: ");
    displayMenu();
    $choice = readline("Enter your choice (a-d): ");

    switch ($choice) {
        case 'a': displayFileType($filename); break;
        case 'b': deleteFile($filename); break;
        case 'c': displayLastAccessTime($filename); break;
        case 'd': echo "Exiting the program.\n"; break;
        default: echo "Invalid choice!\n";
    }
} while ($choice != 'd');
?>

Q.2 Write a Python program to create a Pie plot to get the frequency of

import matplotlib.pyplot as plt
from sklearn.datasets import load_iris
iris = load_iris()
data = pd.DataFrame(data=iris.data,
columns=iris.feature_names)

Q.1 Accept a string from the user and check whether it is a palindrome
 or not (Implement stack operations using array built-in functions). ?

ans-

 <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Palindrome Checker</title>
</head>
<body>
    <h2>Enter a String</h2>
    <form method="post">
        <input type="text" name="inputString" required>
        <input type="submit" value="Check">
    </form>

    <?php
    if ($_POST) {
        $str = $_POST['inputString'];
        $stack = str_split($str);
        $reversed = '';

        while ($char = array_pop($stack)) {
            $reversed .= $char;
        }

        echo $str === $reversed ? "<h3>'$str' is a palindrome.</h3>" : "<h3>'$str' is not a palindrome.</h3>";
    }
    ?>
</body>

Q.1Consider the following entities and their relationship.
Doctor (dno, dname, address, city)
Hospital (hno, hname, hcity)
Doctor-Hospital related with many to one relationship.
Create a RDB in 3NF for above and solve the following.
Write a PHP script to print the Doctor visiting to the Hospital in
tabular format.
Accept hospital name from user ?

ans-

 CREATE TABLE Hospital (
    hno INT PRIMARY KEY,
    hname VARCHAR(100),
    hcity VARCHAR(100)
);

CREATE TABLE Doctor (
    dno INT PRIMARY KEY,
    dname VARCHAR(100),
    address VARCHAR(255),
    city VARCHAR(100),
    hno INT,
    FOREIGN KEY (hno) REFERENCES Hospital(hno)
);

INSERT INTO Hospital (hno, hname, hcity) VALUES

Q.2 View the dataframe, print the shape, number of rows-columns,
data types, feature names and the description of the data and basic
statistical details of the data.(Use iris.csv) ?

ans-

import pandas as pd

df = pd.read_csv('iris.csv')
shape = df.shape
rows, columns = df.shape
data_types = df.dtypes
feature_names = df.columns
description = df.describe()

print("Shape of dataset:", shape)
print("Number of rows:", rows)
print("Number of columns:", columns)
print("Data types:\n", data_types)
print("Feature names:\n", feature_names)
print("Description of dataset:\n", description)
